Text Field

Text fields let users enter and edit text.

  • Material Design guidelines: Text Fields

Import

Design & API Documentation

Selector: mdc-text-field

Properties Description
id: string Unique id of the element.
type: string = 'text' Input type of textfield (e.g.: email, password, date, color).
name: string Name of the textfield.
label: string Shown to the user when there's no focus or values.
value: string The input element's value.
outlined: boolean Set a border around all sides of the text field.
dense: boolean Shrinks the font size and height of the input.
fullwidth: boolean Set to fullwidth textfield.
disabled: boolean Disables the component.
readonly: boolean Whether or not the textfield value is editable.
required: boolean Whether the element is required.
useNativeValidation: boolean Sets whether to check native HTML validity state (true, default) or custom validity state when updating styles (false).
valid: boolean Updates input validity styling using passed value.
empty: boolean Returns whether the control is empty.
autocomplete: string Indicates if the input can be automatically completed by the browser, usually by remembering previous values the user has entered.
tabIndex: number Tab index of the text element.
pattern: string Regular expression that the control's value is checked against.
maxlength: number Maxlength of characters allowed to be entered.
minlength: number Specifies the minimum number of characters that the user can enter.
max: number The maximum numeric value for the input.
min: number The minimum numeric value for this input, which must not be greater than its maximum (max attribute) value.
step: number Works with the min and max attributes to limit the increments at which a numeric or date-time value can be set.
size: number The initial size of the control.
helperText: MdcTextFieldHelperText Reference to input's MdcTextFieldHelperText directive
Methods Description
focus() Sets focus to the input.
Events Description
change(value: string) Emitted whenever the value of the input changes.
blur(value: string) Emitted whenever the input loses focus.

Selector: mdcTextFieldHelperText

Properties Description
validation: boolean Help text can be used to provide additional validation messages.
persistent: boolean Help text will always be visible.

Selector: mdc-icon[mdcTextFieldIcon]

Properties Description
leading: boolean Styles the text field as a text field with a leading icon.
trailing: boolean Styles the text field as a text field with a trailing icon.

Selector: mdc-textarea

Properties Description
rows: number Number of rows for this textarea.
cols: number Number of columns for this textarea.
